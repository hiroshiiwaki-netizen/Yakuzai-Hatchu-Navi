# 実装計画: 起動画面（ローディング）の改善

## 概要
現在のアプリアイコンだけのシンプルなローディング画面から、**アプリの機能や魅力を伝えるスライドショー形式の起動画面**に変更します。
待機時間を「退屈な時間」から「アプリを知る時間」に変え、体感的な待ち時間を短縮します。

## ユーザーレビューが必要な事項
- スライドに表示する日本語メッセージの内容（下記「実装詳細」参照）
- デザインの雰囲気（「かわいく使いやすい」パステルグリーン基調）

## 実装詳細

### 変更するファイル
`index.html` と `styles.html` のみを変更します。GAS側の変更はありません。

### 1. 画面デザイン
- **背景**: パステルグリーンの優しいグラデーション（`#e0f2f1` 〜 `#b9f6ca` 系）
- **カード**: 中央に白背景の角丸カードを配置
- **アニメーション**: スライドが自動でフェードイン・フェードアウト

### 2. スライドの内容（日本語）
以下の4つのメッセージを数秒ごとに切り替えます。絵文字を大きく配置して親しみやすくします。

| No | アイコン | メインメッセージ | サブテキスト |
| :--- | :---: | :--- | :--- |
| **1** | 👋 | **ようこそ！** | 薬剤発注ナビへ<br>チームみんなで在庫管理 |
| **2** | 🔄 | **定期発注に対応** | 「毎週」「毎月」の注文は<br>アプリにおまかせ！ |
| **3** | 🔍 | **カンタン検索** | 患者名や薬剤名で<br>すぐに見つかります |
| **4** | ✅ | **ステータス共有** | 「発注済」「納品済」が<br>ひと目でわかります |

### 3. 下部表示
- **プログレスバー**: 薬のカプセルをモチーフにした、少し遊び心のあるデザイン
- **テキスト**: 「読み込みが遅れています...」ではなく**「準備しています...」**に変更

## 検証計画
### 手動検証
1.  **表示確認**: アプリを起動（リロード）し、スライドがアニメーションするか確認
2.  **文字確認**: 日本語メッセージが正しく表示されるか、読みやすいか確認
3.  **レスポンシブ**: スマホ/PC両方でレイアウトが崩れないか確認
4.  **遷移確認**: 読み込み完了後、スムーズにメイン画面（リスト）に切り替わるか確認
