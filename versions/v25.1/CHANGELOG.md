# CHANGELOG - v25.1.0

## 概要
法人規定デザインガイドライン準拠のため、UIカラーを緑系からオレンジ/赤系に変更。

## 変更内容

### UI改善
- **カラーパレット変更**: 緑系 → オレンジ/赤系（法人規定準拠）
  - プライマリカラー: `#4caf50` → `#E32D00` (OHISAMA ORANGE)
  - ダークカラー: `#2e7d32` → `#d64123` (RED)
  - ライトカラー: `#e8f5e9` → `#fff5f0`
  - グラデーション: 緑系 → オレンジ系
  - 影の色調: 緑系 → オレンジ系

### 変更ファイル
- `styles.html`: CSS変数を全面的に更新

## 影響範囲
- すべての画面のカラー表示が変わります
- 機能面での変更はありません

## 移行手順
1. v25.1フォルダのコードをSRCにコピー
2. `clasp push` でGASに反映
3. GASエディタで「新しいデプロイ」を作成
